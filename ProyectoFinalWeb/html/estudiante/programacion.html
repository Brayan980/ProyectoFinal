Unidad 1: Introducción a la Programación

En esta unidad, exploraremos los conceptos fundamentales de la programación, incluyendo la lógica de programación, variables, y estructuras de control.

Algunos temas:

  • ¿Qué es la programación?
  • Lógica de programación
  • Variables y tipos de datos
  • Estructuras de control: if, else, while
Unidad 2: Estructuras de Datos

En esta unidad, nos adentraremos en las estructuras de datos esenciales para la programación eficiente y la manipulación de información.

Algunos temas:

  • Arrays y listas
  • Pilas y colas
  • Árboles y grafos básicos
Unidad 3: Desarrollo Web

Esta unidad se centra en el desarrollo web, abarcando desde HTML y CSS hasta JavaScript y frameworks modernos.

Algunos temas:

  • HTML5 y estructura de página
  • CSS3 y estilos avanzados
  • JavaScript y manipulación del DOM
  • Frameworks web populares
Unidad 4: Bases de Datos

En esta unidad, exploraremos el mundo de las bases de datos, desde el diseño hasta la manipulación de datos con SQL.

Algunos temas:

  • Diseño de bases de datos
  • SQL y consultas
  • Gestión de bases de datos
